Purpose

This page is the chronological record of major wiki evolution milestones.

It does not replace git history. It gives humans and agents a readable timeline of the wiki’s growth.

Major milestones

[2026-04-06] source-doc gap fill phase

  • Added wiki-gap-review-2026-04-06 to record the source-doc versus wiki comparison and prioritized gaps.

[2026-04-05] anatomy-layer phase

  • Added anatomy pages for app shell, tablet/mobile/controller now-playing, kiosk shell, config, diagnostics, library health, and theme.

  • Added queue/playback model, playback authority, queue wizard internals, controller queue interface, route ownership, fragile behavior ownership, and runbook trio.
  • Added wiki-structure-notes.

[2026-04-05] rendered-wiki usability phase

  • Grouped rendered nav, fixed broken links, polished hub and branch pages, and added repo coverage notes.

[2026-04-05] config and API branch expansion phase

  • Expanded config into a real subsystem branch and created the first API branch pages.

[2026-04-05] interface and surface coverage phase

  • Established interface-first organization and built out major surface pages across desktop, tablet, phone, display, kiosk, config/diagnostics, and integration areas.

[2026-04-05] framing and recovery phase

  • Reframed the wiki around now-playing as a moOde-centered enhancement ecosystem and recovered the README after accidental overwrite.

Current status

The wiki has moved from broad coverage into a more explicit knowledge system with browse/search, anatomy, synthesis, and operations layers. That progression is itself useful context for future agents.
