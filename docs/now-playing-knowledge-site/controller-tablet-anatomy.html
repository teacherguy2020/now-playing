controller tablet anatomy

Purpose

This page documents the internal anatomy of now-playing/controller-tablet.html.

It is a multi-region orchestration shell with now-playing header behavior, integrated transport, quick search, category navigation, recent-rail browsing, tablet-specific next-up/action regions, pane embedding, modal workflows, and heavy device-preset/layout logic.

Main anatomical regions

  1. kiosk/editor control strip
  2. now-playing header card
  3. integrated transport controls
  4. quick search strip
  5. library/category list
  6. recent-content region
  7. tablet next-up / action bars
  8. kiosk pane iframe region
  9. album / artist / audio-info modal family
  10. device-preset and layout-profile logic

Kiosk/editor control strip

Anchored by #kioskCtl, #kioskSizeSel, #kioskPushBtn, and #kioskPushStat. This is the tablet page’s local control strip for sizing/pushing kiosk-like output in certain contexts.

Now-playing header card

Anchored by #npCard, .npTap, .np, #npArtWrap, #art, #npPersonnelBody, #line1, #line2, #line3, and #line2Stars. This is the main top-left playback summary region.

Integrated transport controls

Anchored by #npCtrls and data-np-act buttons for repeat, previous, play/pause, next, and shuffle. This is one of the clearest reasons controller-tablet.html is a real controller shell rather than just a navigation page.

Quick search strip

Anchored by #homeQuickSearch, #homeQuickKb, #homeQuickClear, #homeQuickResults, and .quickWrap.

Library/category list

Anchored by #libraryList, .row, and .rowWrap. Visible row targets include Playlists, Artists, Albums, Genres, Podcasts, Radio, Queue Wizard, Live Queue, Visualizer, and Settings.

The Live Queue row is richer and includes direct row actions for Vibe, Crop, and Clear.

Recent-content region

Anchored by #recentTitle, #recentRail, #recentPeekRail, #railModeBtn, #ipadRecentGrid, and the individual #ipadRail* elements. This region can behave as a single rail, a peek rail, or a multi-pane recent grid depending on profile/layout/device mode.

Tablet next-up / action bars

Anchored by #tabletTopCards, #tabletNextUpBar, #tabletNextUpText, #tabletActionBar, #tabletQueuePos, #tabletAudioInfoBtn, plus the inline next-up anchors #tabletNextUpInline, #tabletNextUpArtInline, and #tabletNextUpTextInline.

Kiosk pane iframe region

Anchored by #kioskPane and #kioskPaneFrame. This is one of the tablet page’s most important orchestration features and overlaps strongly with embedded-pane and kiosk routing behavior.

Album / artist / audio-info modal family

Includes #audioInfoModal, #recentAlbumModal, #recentArtistModal, and related action buttons like #recentAlbumAppendBtn, #recentAlbumCropBtn, #recentAlbumReplaceBtn, and #recentArtistShuffleBtn.

Device-preset and layout-profile logic

controller-tablet.html contains substantial logic for device presets and layout profiles such as tablet, ipad11, ipadModern, ipadAir, tabletLandscape, kiosk1280, and kioskEditor. This logic materially changes grid structure, card sizing, recent-rail layout, art/control placement, next-up/action-bar visibility, embedded pane behavior, and kiosk/editor behavior.

Practical “where do I start?” map

  • Now-playing header: start with #npCard, .np, #npArtWrap, and the line fields.
  • Tablet next-up area: start with #tabletNextUpBar, #tabletNextUpText, and #tabletNextUpInline.
  • Quick search: start with .quickWrap, #homeQuickSearch, and #homeQuickResults.
  • Library/category list: start with #libraryList, .row, and the Live Queue row action wiring.
  • Recent rails: start with #recentRail, #recentPeekRail, #ipadRecentGrid, and layout-profile logic.
  • Right-pane embedded behavior: start with #kioskPane, #kioskPaneFrame, and the embedded-pane/kiosk routing companions.
  • Album/artist/audio info modals: start with the relevant modal IDs and related action buttons.

Current status

This is the second real page-anatomy reference in the wiki. Its main job is making explicit that controller-tablet.html is not just a tablet-themed controller page, but a multi-region orchestration shell.
