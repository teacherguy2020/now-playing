api endpoint catalog

Purpose

This page is the first source-driven endpoint catalog for the now-playing app-host API.

It is based directly on source-visible routes, mainly in moode-nowplaying-api.mjs and src/routes/*.mjs.

Scope and proof model

This page is strong on endpoint existence and owner-file visibility, but not yet a full request/response contract spec for every route.
